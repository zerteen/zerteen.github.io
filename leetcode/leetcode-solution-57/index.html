  • 若interval在当前区间的右边，将当前区间添加到ans中
  • 若interval在当前区间的左边，判断是否已经插入过interval
    • 若没插入过interval，插入interval，修改插入标记，插入当前区域
    • 若已插入过interval，插入当前区域
  • 若interval和当前区间有交集，将interval.start 取interval和当前区间的最小值，interval.end 取interval和当前区域的最大值，继续循环。
