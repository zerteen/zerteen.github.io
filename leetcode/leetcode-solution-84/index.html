Solution 1:

  • 遍历一遍数组，依次比较相邻的两个元素，同时，设定一个tag isSame 来保存上两个元素是否一样
  • 如果上两个元素一样，指针p q 往后移一位，继续
  • 如果两个元素不一样，将p加入到待返回的列表中，继续
  • 循环结束时，isSame中保存了最后一次的比较情况，如果最后一次结果一样，返回列表的最后需要加上null 如果最后一次结果不一样，需要将p之后的元素都添加到返回列表中。
